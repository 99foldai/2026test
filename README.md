# 2026test

A Next.js application with custom brand styling.

## Brand Colors

- **Primary:** #284e36 (Forest Green)
- **Background:** #efefef (Light Gray)
- **Font:** #151515 (Near Black)
- **Card:** #F3F1DA (Cream)
- **Button:** #151515 (Near Black)

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Available Components

This template includes pre-styled Tailwind components:

- `.btn-primary` - Primary button (dark)
- `.btn-secondary` - Secondary button (green)
- `.btn-outline` - Outline button
- `.card` - Card component with cream background
- `.input` - Styled form input
- `.badge` - Badge component

## Color Usage

Use Tailwind classes with your brand colors:

- `bg-primary-800` - Primary green
- `bg-neutral-900` - Dark button color
- `bg-card-300` - Card background
- `bg-neutral-100` - Page background
- `text-neutral-900` - Body text color

## Deploy on Vercel

1. Push your code to GitHub
2. Import your repository in Vercel
3. Deploy with one click

Check out the [Next.js deployment documentation](https://nextjs.org/docs/app/building-your-application/deploying) for more details.
